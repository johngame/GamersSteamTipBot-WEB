Quick Start

  1. Join the Steam Group
  2. Send +register to the bot
  3. Start Tipping!
Once you register you can manually deposit Gamerscoins into the given address or wait for someone to tip you.

List of Commands

+register

The register command will only be used once upon joining the group. It generates a personal address bound to your Steam account which you will use to deposit Gamerscoins into, also receive and send tips.

+add

Sets up a Moolah request to add funds to your tipping account

This exmaple provides you with an address to send your 200 Gamerscoins to
+add 200
You will have 30 minutes before the address expires.
+tip

Use this command to send a tip to any user.

This exmaple sends 200 Gamerscoins to the user "shibe"
+tip shibe 200
This exmaple sends 200 Gamerscoins to the user "shibe" with your message attached to it
+tip shibe 200 "GG!"
This example sends 200 Gamerscoins to the user "shibe" anonymously
+tip shibe 200 hideme
